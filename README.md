# Currency Converter 

A simple and responsive Currency Converter web application built using **HTML, CSS, and JavaScript**.  
It fetches real-time exchange rates using an external API and converts currencies instantly.

---

## Features

- Convert currencies using real-time exchange rates
- Supports multiple international currencies
- Dynamic currency dropdown (loaded from API)
- Input validation for invalid or empty amounts
- Clean and user-friendly UI

---

## Tech Stack

- **HTML** – Structure  
- **CSS** – Styling and layout  
- **JavaScript** – Logic, DOM manipulation, API handling  
- **ExchangeRate API** – Real-time currency data
